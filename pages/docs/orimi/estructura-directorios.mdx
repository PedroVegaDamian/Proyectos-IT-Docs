## src

Es el directorio principal de la aplicación.

### assets

Es el directorio donde se guandan los recursos.

### components

Es el directorio donde se crean los componentes.

### firebase

Es el directorio de configuración de Firebase.

### hooks

Es el directorio donde se crean los hooks.

### layouts

Es el directorio donde se crean los layouts.

### pages

Es el directorio donde se crean las páginas.

### router

Es el directorio donde se configuran las rutas.

### services

Es el directorio donde se crean los servicios que interactuan con firebase.

### store

Es el directorio donde se maneja los estados globales.

### utils

Es el directorio donde se crean los métodos y funciones auxiliares.
